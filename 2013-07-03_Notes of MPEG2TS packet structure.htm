How to resolve the MPEG2TS packet? First of all is to find PAT. PAT include the information of PMT . And the PMT include the information of video/audio PID. Finally you will know the video/audio PID via PMT table. So you can assemble the multiple video/audio PES packet to a video frame or audio via the video/audio PID.

TS Header Properties

syntax value describe
sync_byte 0x47 (71) sync bytes. always 0x47
transport_error_indicator 0 or 1 when error is occurred it's specified 1
payload_unit_start_indicator 0 or 1 It's specified 1 if is the start packet of multiple PES packet.
transport_priority 0 or 1 don't care
PID 13 bit number PAT always 0x00.
transport_scrambling_control 0 or 1 don't care
adaptation_field_control 0x02 or 0x01 or 0x03 if include adaptation_field then the second bit is 1. if include payload then the first bit is 1
continuity_counter 4bit number. 0x00 ~ 0x15 the serial number of PES packet
