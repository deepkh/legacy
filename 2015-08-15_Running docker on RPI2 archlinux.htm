So I have try to find other solution. After I'm search for a while. I find that ArchLinux has been succeeded porting docker 32bit arm to their binary tree. So I'm happyly to install docker on archlinux. Archlinux clear install size 800MB are more fat than Debian Jessie 280MB. 

After you install docker succeeded on your RPi2. You should pulling the following docker image on your RPi2. And I have only try "mazzolino/armhf-ubuntu:14.04". The armhf-ubuntu:14.04 seem running well on docker just work like x86_64 ubuntu:14.04. The image size about 280MB. And also I have try to install some build-esstinal.
